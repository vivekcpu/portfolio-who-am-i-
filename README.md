# VivekOS | Personal Portfolio 🚀

Welcome to the source code of my personal portfolio. Designed with a brutalist aesthetic, this static site serves as a showcase of my full-stack engineering experience, competitive programming achievements, and latest projects.

🔗 **Live Site:** [https://vivekcpu.github.io/portfolio-who-am-i-/]

## ✨ Features

* **Brutalist UI:** High-contrast, bold borders, and a stark aesthetic optimized for readability and speed.
* **Interactive Web Terminal:** A custom-built CLI at the bottom of the page that accepts commands like `whoami`, `skills`, `cp`, and `calc`.
* **The Digital Rain:** A hidden `matrix` command that triggers a custom HTML5 Canvas digital rain effect directly inside the terminal.
* **Performance First:** Pure HTML, CSS, and Vanilla JavaScript with zero heavy framework bloat for near-instant load times.
* **Responsive Layout:** CSS Grid and Flexbox architecture that scales perfectly from 4k monitors down to mobile screens.

## 🛠️ Tech Stack

* **Markup & Styling:** HTML5, CSS3 (Custom Properties, CSS Grid, Flexbox)
* **Logic & Interaction:** Vanilla JavaScript (ES6+)
* **Graphics:** HTML5 `<canvas>` API (for the Matrix effect)
* **Typography:** Space Grotesk & Fira Code (via Google Fonts)
* **Icons:** FontAwesome 6
